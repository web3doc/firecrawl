## 设置您的开发环境

了解如何本地更新文档并将其部署到公共服务器。

### 编辑和预览

<AccordionGroup>
  <Accordion icon="github" title="在本地克隆您的文档">
    在入门过程中，我们在您的Github上创建了一个包含您文档内容的仓库。您可以在我们的
    [仪表板](https://dashboard.mintlify.com)上找到这个仓库。要本地克隆该仓库，请按照以下
    [说明](https://docs.github.com/en/repositories/creating-and-managing-repositories/cloning-a-repository)
    在终端中操作。
  </Accordion>
  <Accordion icon="rectangle-terminal" title="预览更改">
    预览帮助您确保更改按预期显示。我们构建了一个命令行界面来本地渲染这些更改。1. 安装
    [Mintlify CLI](https://www.npmjs.com/package/mintlify)，以使用以下命令本地预览文档更改：``` npm i -g mintlify ```
    2. 在您的文档根目录（即`mint.json`所在位置）运行以下命令：``` mintlify dev ```
  </Accordion>
</AccordionGroup>

### 部署您的更改

<AccordionGroup>

<Accordion icon="message-bot" title="安装我们的GitHub应用">
  我们的GitHub应用会自动将您的更改部署到您的文档网站，因此您无需自行管理部署。您可以在您的
  [仪表板](https://dashboard.mintlify.com)上找到安装链接。一旦机器人成功安装，应该在仓库的提交哈希旁边有一个勾选标记。
</Accordion>
<Accordion icon="rocket" title="推送您的更改">
  [提交并推送您的更改至Git](https://docs.github.com/en/get-started/using-git/pushing-commits-to-a-remote-repository#about-git-push)
  以便您的更改在文档网站上更新。如果您推送后没有看到GitHub应用成功部署您的更改，您也可以通过我们的
  [仪表板](https://dashboard.mintlify.com)手动更新您的文档。
</Accordion>

</AccordionGroup>
