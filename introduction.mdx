## 欢迎使用 Firecrawl

[Firecrawl](https://firecrawl.dev?ref=github) 是一项 API 服务，可接受 URL，对其进行爬取，并将其转换为干净的 Markdown。我们会爬取所有可访问的子页面，并为每个页面提供干净的 Markdown。无需站点地图。


## 如何使用？

我们提供易于使用的 API 和托管版本。您可以在 [这里](https://firecrawl.dev/playground) 找到 playground 和文档。您也可以自行托管后端。

### API Key

要使用 API，您需要在 [Firecrawl](https://firecrawl.dev) 上注册并获取一个 API key。


### 功能

- [**抓取**](#scraping): 抓取 URL 并获取其内容（Markdown、结构化数据、截图、HTML）以供 LLM 使用。
- [**爬取**](#crawling): 抓取网页的所有 URL 并返回可供 LLM 使用的内容。
- [**映射**](/features/map): 输入一个网站并获取所有网站 URL - 非常快速。
- [**提取**](/features/extract): 使用 AI 从单个页面、多个页面或整个网站中获取结构化数据。
### 强大功能
- **LLM 准备格式**: Markdown、结构化数据、截图、HTML、链接、元数据
- **复杂功能**: 代理、反爬虫机制、动态内容（JS 渲染）、输出解析、编排
- **可定制性**: 排除标签、在自定义标头后爬取授权墙、最大爬取深度等...
- **媒体解析**: PDF、DOCX、图像。
- **可靠性优先**: 旨在获取您需要的数据 - 无论多么困难。
- **操作**: 点击、滚动、输入、等待等操作，然后提取数据。


您可以在我们 [文档](https://docs.firecrawl.dev) 中找到 Firecrawl 的所有功能以及如何使用它们。
