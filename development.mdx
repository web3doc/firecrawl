<Info>
  **先决条件** 您应该已经安装了 Node.js（版本 18.10.0 或更高）。
</Info>

步骤 1. 在您的操作系统上安装 Mintlify：

<CodeGroup>

```bash npm
npm i -g mintlify
```

```bash yarn
yarn global add mintlify
```

</CodeGroup>

步骤 2. 转到文档所在的位置（您可以在其中找到 `mint.json`），然后运行以下命令：

```bash
mintlify dev
```

文档网站现在可在 `http://localhost:3000` 访问。

### 自定义端口

Mintlify 默认使用端口 3000。您可以使用 `--port` 标志来自定义 Mintlify 运行的端口。例如，使用以下命令在端口 3333 运行：

```bash
mintlify dev --port 3333
```

如果尝试在已被占用的端口上运行 Mintlify，您将看到如下错误：

```md
Error: listen EADDRINUSE: address already in use :::3000
```

## Mintlify 版本

每个 CLI 都链接到特定版本的 Mintlify。如果本地网站与生产环境不同，请更新 CLI。

<CodeGroup>

```bash npm
npm i -g mintlify@latest
```

```bash yarn
yarn global upgrade mintlify
```

</CodeGroup>

## 故障排除

以下是解决使用 CLI 时常见问题的方法。

<AccordionGroup>
  <Accordion title="Mintlify 无法加载">
    更新到 Node v18。运行 `mintlify install` 然后重试。
  </Accordion>
  <Accordion title="Windows 上没有这样的文件或目录">
去 `C:/Users/Username/.mintlify/` 目录并删除 `mint` 文件夹。然后在此位置打开 Git Bash 并运行 `git clone https://github.com/mintlify/mint.git`。
再次执行步骤 3。
  </Accordion>
  <Accordion title="遇到未知错误">
    尝试导航到设备的根目录并删除 ~/.mintlify 文件夹。然后再次运行 `mintlify dev`。
  </Accordion>
</AccordionGroup>
