## 图片

### 使用Markdown

[Markdown语法](https://www.markdownguide.org/basic-syntax/#images)允许你使用以下代码添加图片：

```md
![标题](/path/image.jpg)
```

注意，图片文件的大小必须小于5MB。否则，我们建议将图片托管在像[Cloudinary](https://cloudinary.com/)或[S3](https://aws.amazon.com/s3/)这样的服务上。然后你可以使用该URL进行嵌入。

### 使用嵌入

为了更灵活地使用图片，你也可以使用[嵌入](/writing-content/embed)来添加图片：

```html
<img height="200" src="/path/image.jpg" />
```

### iFrames

在文档内加载另一个HTML页面。最常用于嵌入视频。

```html
<iframe src="https://www.youtube.com/embed/4KzFe50RQkQ"> </iframe>
```
