## 基础

### 内联代码

用反引号 (`) 包围 `单词` 或 `短语` 来表示代码。

```
用反引号 (`) 包围 `单词` 或 `短语` 来表示代码。
```

### 代码块

使用 [带围栏的代码块](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks)，通过在代码周围添加三个反引号，并在前导反引号之后添加代码片段的编程语言以获得语法高亮。可选地，你也可以在编程语言名称后写上你的代码名称。

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
