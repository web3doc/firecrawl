## 设置

安装我们的python依赖，包括groq和firecrawl-py。

```bash
pip install groq firecrawl-py
```

## 获取您的Groq和Firecrawl API密钥

要使用Groq和Firecrawl，您需要获取您的API密钥。您可以从[这里](https://groq.com)获取您的Groq API密钥，从[这里](https://firecrawl.dev)获取您的Firecrawl API密钥。

## 使用Firecrawl加载网站

为了能够从网站页面获取所有数据并确保其处于最干净的格式，我们将使用[Firecrawl](https://firecrawl.dev)。它处理绕过JS阻止的网站，提取主要内容，并以LLM可读格式输出以提高准确性。

以下是我们如何使用Firecrawl抓取网站URL的方法。我们还将设置`pageOptions`以仅提取网站页面的主要内容（`onlyMainContent: True`）- 排除导航、页脚等。

```python
from firecrawl import FirecrawlApp  # 导入FireCrawlLoader

url = "https://about.fb.com/news/2024/04/introducing-our-open-mixed-reality-ecosystem/"

firecrawl = FirecrawlApp(
    api_key="fc-YOUR_FIRECRAWL_API_KEY",
)
page_content = firecrawl.scrape_url(url=url,  # 目标URL进行爬取
    params={
        "pageOptions":{
            "onlyMainContent": True # 忽略导航、页脚等
        }
    })
print(page_content)
```

完美，现在我们有了来自网站的干净数据 - 准备被提供给LLM进行数据提取。

## 提取与生成

现在我们已经拥有了网站数据，让我们使用Groq来提取我们需要的信息。我们将使用Groq Llama 3模型在JSON模式下，并从页面内容中挑选出某些字段。

我们在这个示例中使用LLama 3 8b模型。请随意使用更大的模型以获得更好的结果。

```python
import json
from groq import Groq

client = Groq(
    api_key="gsk_YOUR_GROQ_API_KEY",  # 注意：用您实际的Groq API密钥替换'API_KEY'
)

# 在这里我们定义想要从页面内容中提取的字段
extract = ["summary","date","companies_building_with_quest","title_of_the_article","people_testimonials"]

completion = client.chat.completions.create(
    model="llama3-8b-8192",
    messages=[
        {
            "role": "system",
            "content": "你是一名法律顾问，负责从文档中提取信息并以JSON格式呈现。"
        },
        {
            "role": "user",
            # 在这里我们传递页面内容和想要提取的字段
            "content": f"从提供的文档中提取以下信息:\Page content:

{page_content}

要提取的信息: {extract}"
        }
    ],
    temperature=0,
    max_tokens=1024,
    top_p=1,
    stream=False,
    stop=None,
    # 我们设置响应格式为JSON对象
    response_format={"type": "json_object"}
)


# 漂亮打印JSON响应
dataExtracted = json.dumps(str(completion.choices[0].message.content), indent=4)

print(dataExtracted)
```
