> 注意：此示例正在使用[v0版本的Firecrawl API](/v0/introduction)。你可以安装Python SDK的0.0.20版本或Node SDK的0.0.36版本。

> [GitHub仓库 (https://github.com/e2b-dev/e2b-cookbook) ](https://github.com/e2b-dev/e2b-cookbook/tree/main/examples/scrape-and-analyze-airbnb-data-with-firecrawl)

这个示例展示了如何使用[Firecrawl](https://www.firecrawl.dev/)和E2B的[代码解释器SDK](https://github.com/e2b-dev/code-interpreter)来抓取和分析Airbnb数据。

我们不会编写任何分析代码，而是让Anthropic的AI为我们编写。我们将使用E2B的代码解释器SDK安全地运行代码并获取结果。

## 前提条件

- 你的计算机上安装了Node.js
- 一个E2B API密钥
- 一个Firecrawl API密钥
- 一个Anthropic API密钥

## 设置与运行

### 1. 克隆仓库

这是此示例的[GitHub仓库](https://github.com/e2b-dev/e2b-cookbook/tree/main/examples/scrape-and-analyze-airbnb-data-with-firecrawl)。

```
git clone https://github.com/e2b-dev/e2b-cookbook.git
cd e2b-cookbook/examples/scrape-and-analyze-airbnb-data-with-firecrawl
```

### 2. 安装依赖项

```
npm install
```

### 3. 设置`.env`文件

1. 将`.env.template`复制为`.env`
2. 获取[E2B API密钥](https://e2b.dev/docs/getting-started/api-key)
3. 获取[Firecrawl API密钥](https://firecrawl.dev)
4. 获取[Anthropic API密钥](https://anthropic.com)

### 4. 运行示例

```
npm run start
```
